This repository contains experiments performed and their results, as included in the EPIA manuscript. 

Whereby: we have three folders (scrapping, working_dir, and roles-enhanced2). 

**scrapping folder:** contains the initial context gathered when we obtained job advertisements to generate role profiles for LLM variants. 

**working_dir folder:** contains scripts for generating the role profiles and assessing their strengths as we run multiple iterations to assess the semantic similarity of the generated profiles.

**roles-enhanced2 folder:** contains scripts for prompting LLMs and analyzing generated responses (i.e., clustering and the Jaccard similarity index).
